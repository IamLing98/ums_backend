# University Management System

## About The Project

This is my final year project in my university. An university information management system. 

### Built With
 
* [Spring](https://spring.io/)
* [FCM](https://firebase.google.com/docs/cloud-messaging/)
* [Websocket](https:) 

<!-- GETTING STARTED -->
## Getting Started


``` bash
# build with gradle
gradlew clean build

# run with gradle
gradlew bootRun
 
```
